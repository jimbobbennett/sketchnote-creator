# Examples

Finished sketchnotes produced by this skill (4K, 16:9), self-contained PNGs.

- **`keynote-arize-observe-2026.png`** — *The Future of AI Agents* (Arize Observe 2026 founders' keynote).
  12 sections, speaker portraits (Aparna Dhinakaran, Jason Lopatecki), slide-derived sketches.
- **`openai-feedback-chatgpt-scale.png`** — *Harnessing User Feedback at ChatGPT Scale* (OpenAI, Stuart Sy).
  12 sections, speaker portrait, 4 selective slide sketches, "vibes → evidence → action" arc.

Both were generated end-to-end via `SKILL.md` (with `--design design/arize`) and fact-checked against
the source video with the Gemini + Codex validators. Run without `--design` to get the generic
`design/default` look instead.
